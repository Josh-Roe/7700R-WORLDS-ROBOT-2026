This is the 7700R CAD that was used for the 2026 VEX World Championship. This CAD was designed on Autodesk Inventor and uploaded onto GitHub because Autodesk Inventor does not have a cloud system. This CAD contains everything down to the screws and nuts of the robot and took approximately 2 weeks to make.

This CAD model is precise enough to make a 98% complete copy of our robot that we brought to the World Championship.

Systems were designed like:
- All 12 custom plastic pieces
- Full drivebase
- Structure for the S Robot design
- Gearing and perfect meshing
- Piston mounting for optimal compression and actuation length
- Ziptie compression on the plastic curves
- Preroller subsystem
- Crossbraces
- Middle goal descore
- Hood rollers
- Core intake system
- Odometry tracking wheels
